- 커뮤니티형 콘텐츠 공유를 원하는 사람들 및 블로그 글쓰기를 즐기거나, 다양한 주제의 글을 읽고 싶은 사용자.


## 기술 도구
- **Backend : Spring Boot, Java, JPA, Gradle**
- **Frontend : React, TypeScript, CSS, HTML**
- **Database : MySQL, MariaDB**
- **IDE : VScode**
- **형상 관리 : Github**


## 기능
- 블로그 게시물 작성, 수정, 삭제 가능
- 게시물별 댓글 및 좋아요 기능 제공
- 회원가입, 로그인, 로그아웃 기능
- 최신 게시물 목록 및 이미지 첨부 지원
- 주간 Top 3 게시물 제공 (댓글 수, 좋아요 수, 조회 수 기준 인기 게시물 3개)
- 게시물 검색 기능
- 인기 검색어 목록 제공 (검색 결과 기반)
